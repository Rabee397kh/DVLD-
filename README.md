# DVLD System (Driving & Vehicle Licensing Department)

## 📌 Project Overview
The DVLD system is a robust desktop application designed to manage the entire lifecycle of driving licenses, from application and testing to issuance and renewal.

## 🚀 Features
- **Comprehensive People Management:** Centralized person records used across all modules.
- **License Management:** Issuance, Renewals, Replacements, and Detained Licenses.
- **Testing System:** Multi-stage test scheduling and result tracking.

## 🛠️ Technical Implementation
- **C# & ADO.NET:** Clean, handwritten data access without heavy ORMs.
- **SQL Server:** Relational database design with optimized queries.
- **3-Layer Architecture:** Decoupled UI, Logic, and Data.
